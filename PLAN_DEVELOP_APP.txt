Lineamientos escenciales
- Aplicacion de escritorio multiplataforma (Windows/macOS/Linux)
- Usar siempre buenas practicas
- Proveer de atajos desde el teclado, el usuario debe poder manejar la aplicacion sin usar el mouse

Funcionalidades:
- 100% offline
- Capacidad de sincronizar datos con un servidor cuando hay conexion a internet
- Capadidad de leer inputs por dispositivos externos como maquina lectora de cГіdigos de barras
- Servidor para sincronizacion de datos
- API para comunicacion entre la aplicacion y el servidor


Infraestructura:
- Base de datos local (los productos tienen que estar respaldados, para ante cualquier error nunca perder los datos)
- Tablas:
    - usuarios (id_usuario, nombre, password, rol)
    - productos (id_producto, nombre, categoria, precio, stock, codigo de barras, descripcion, id_proveedor)
    - proveedores (id_proveedor, nombre, telefono, direccion, email)
    - ventas (id_venta, id_producto, cantidad, precio, fecha, id_usuario)
    - egresos (id_egreso, descripcion, monto, fecha, id_usuario)
    - logs (id_log, descripcion, fecha, id_usuario)
